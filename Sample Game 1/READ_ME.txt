How to set up the game:

	1. Open the game folder called: "THE CASE OF THE FISH"
	2. Download the entier folder(Yes, all of the images, text files, and python files are important)
	3. Extract the downloaded file to your PycharmProjects folder, which on Windows at least should be 
	   C:\Users\(your account name)\PycharmProjects
	4. Open pycharm
	5. Open the game directory called "THE CASE OF THE FISH"
	6. Open the python file
	7. Press the green arrow to play
	8. Have fun, follow the instructions on the dialog boxes
